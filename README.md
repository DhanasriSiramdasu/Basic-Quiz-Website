# 🧠 Quiz Web Application

A simple and interactive **Quiz Application** built using **HTML, CSS, and JavaScript**. The application presents multiple-choice questions, provides instant feedback on selected answers, keeps track of the user's score, and displays the final result at the end of the quiz.

---

## 🚀 Features

- ▶️ Start the quiz with a single click
- ❓ Multiple-choice questions
- ✅ Instant feedback for correct answers
- ❌ Highlights incorrect answers
- 🔒 Prevents changing an answer after selection
- ➡️ Navigate through questions using the **Next** button
- 🏆 Displays the final score after completing the quiz
- 📱 Simple and responsive user interface

---

## 🎯 How to Use

1. Click the **Start Quiz** button.
2. Read the question displayed on the screen.
3. Select one of the available answers.
4. The selected answer is immediately highlighted:
   - 🟢 Green for the correct answer.
   - 🔴 Red for an incorrect answer.
5. Click **Next** to proceed to the following question.
6. After the last question, click **Check Score** to view your final score.

---

## ⚙️ Application Logic

- Questions and answers are stored in a JavaScript array.
- Questions are displayed dynamically.
- Buttons are generated dynamically for each question.
- The selected answer is validated instantly.
- The score increases for every correct answer.
- At the end of the quiz, the total score is displayed.
